# Documentation archive

Archive는 삭제 대체물이 아니라 완료된 기록의 보존 위치입니다. 현재 구현 판단에는 `docs/product_bible/`, `docs/wiki/`, `docs/design/`, `docs/governance/`를 사용합니다.

| Directory | Content |
| --- | --- |
| `design-audits/`, `design-prompts/`, `design-migrations/` | 완료된 Figma 감사·작업 지시·병합 이력 |
| `notion-audits/`, `notion-exports/` | 완료된 동기화 감사와 원본 Export |
| `documentation-plans/`, `planning-history/` | 종료된 문서/구현 계획 |
| `generated-reports/` | 재생성 가능한 화면·도구 산출물 |
| `project-history/`, `github-templates/` | 과거 회의·운영 이력과 비활성 템플릿 |

이동한 Markdown은 상태·이동일·사유·대체 정본·원래 경로를 기록합니다. `notion-exports/`는 원본 스냅샷 무결성을 위해 개별 파일의 본문을 재작성하지 않으며, 이 README가 해당 Export 묶음의 Archive 메타데이터입니다.
