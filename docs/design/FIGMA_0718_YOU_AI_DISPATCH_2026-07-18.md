## STEP A — 로고 복사 (당신 · 5분)

1. 이 링크 클릭 → 헤더/로고가 선택된 채로 열림

2. 왼쪽 레이어에서 **`ASAK LOGO`** (또는 가운데 보이는 ASAK 글자 로고) 클릭  
3. `Ctrl+C` 복사  
4. 0718 파일 열기 → 왼쪽 페이지 **`03-C. Components / Kiosk`** 로 이동  
5. 빈 곳 클릭 후 `Ctrl+V` 붙여넣기  
6. 채팅에: **`로고 붙였어`**

(이미 AI가 쑥색→라임 바인딩은 일부 해 둠. 원본 워드마크를 붙이면 더 정확해짐.)
