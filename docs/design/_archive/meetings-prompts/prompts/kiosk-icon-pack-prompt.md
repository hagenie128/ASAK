> **용도**: 터치스크린 키오스크 UI용 일관된 벡터 스타일 아이콘 16개 시트 생성  
> **기준 DS**: [kiosk-design-system-index.md](../kiosk-design-system-index.md) · **화면 매핑**: [screens.json](../../screens/screens.json) SCR-001~021  
> **참고 형식**: 스터디 카페 키오스크 아이콘 팩 프롬프트 (동일 섹션 구조)

## Version A — DS-01 Fresh Greens (프로젝트 기본 팔레트)

아래 블록 전체를 이미지 생성 도구에 복사해 사용합니다.

```
ASAK Salady 셀프 주문 키오스크 UI에 사용할 현대적인 아이콘 팩을 디자인해줘.

터치스크린 키오스크 인터페이스용으로 일관성 있는 벡터 스타일 아이콘 16개를 만들어줘.

전체 스타일:
- 미니멀한 플랫 벡터 아이콘
- 깔끔한 기하학적 형태
- 단순한 면 형태와 정돈된 라인 사용
- 신선한 샐러드·건강식 키오스크에 어울리는 밝고 친근한 분위기
- 프레시·아삭·건강한 브랜드 무드 (DS-01 Fresh Greens)
- 모든 아이콘은 같은 디자인 언어와 같은 시각적 무게감을 가져야 함

색상 팔레트 (DS-01 Fresh Greens):
- 프레시 그린 (Primary) #16A34A
- 딥 그린 (Pressed/강조) #15803D
- 라이트 그린 (선택·칩 배경) #DCFCE7
- 액센트 라임 (완료·뱃지) #A3E635
- 크림 배경 톤 #FFFDF3
- 화이트 (서피스) #FFFFFF
- 다크 그레이 (아이콘 본체·라인) #1F2937
- 뮤트 그레이 (보조) #6B7280

배경:
- 투명 배경 또는 크림색(#FFFDF3) 단색 배경
- 복잡한 배경·그라데이션·사진 사용 금지

선 스타일:
- 일정한 중간 굵기의 선 (약 2px 상당, 48px 기준)
- 둥근 선 끝 (round cap)
- 둥근 모서리 (round join)
- 모든 아이콘의 선 굵기와 디테일 수준을 동일하게 유지

중요한 시각 규칙:
- 테두리 효과 없음
- 그림자 효과 없음
- 뉴모피즘 효과 없음
- 3D 효과 없음
- 아이콘 컨테이너 박스 없음
- 텍스트 없음
- 숫자 없음
- 글자 없음
- 워터마크 없음
- 로고 없음
- ASAK·Salady 브랜드 워드마크 없음
- 모든 아이콘은 각자의 보이지 않는 정사각형 영역 중앙에 배치
- 아이콘마다 크기와 비율을 균일하게 유지 (터치 타깃 44×44px 이상 대응)
- 작은 크기로 보아도 명확하게 인식 가능해야 함

배치 규칙:
- 16개 아이콘을 4행 4열 그리드로 배치
- 상단, 하단, 좌측, 우측의 바깥 여백을 모두 동일하게 맞춤
- 행 간격과 열 간격을 모두 동일하게 맞춤
- 전체 아이콘 시트가 균형 있고 정돈되어 보여야 함
- 각 아이콘은 같은 크기의 보이지 않는 셀 안에 중앙 정렬

만들 아이콘 목록 (SCR·시나리오 기준):
1. 홈·매장/포장 — 키오스크 시작·브랜드·먹고가기/포장 선택 통합 (SCR-001, SCR-002 병합)
2. 메뉴 카테고리 — 샐러드·랩·음료 등 카테고리 탭 (SCR-003)
3. 옵션·커스터마이즈 — 베이스·드레싱·토핑·재료 제외·알레르기·칼로리 (SCR-004)
4. 품절 — SOLD OUT·재료 품절 뱃지 (SCR-003, SC-003)
5. 장바구니·주문확인 — 담은 메뉴·수량·컨펌 팝업 (SCR-005, SCR-006 병합)
6. 카드 결제 — 신용·체크카드 결제·로딩 (SCR-007, SC-004)
7. 모바일·QR 결제 — 간편결제·QR 스캔 결제 (SCR-007)
8. 결제 실패·재시도 — 팝업/토스트 오버레이 (SCR-012)
9. 포인트·스탬프 적립 — 멤버십 스탬프 1회 확인·적립 (SCR-021, SC-006, Phase 2)
10. 쿠폰 할인 — 쿠폰 QR·바코드 스캔 할인 (SCR-021, SC-016, Phase 2)
11. 주문 완료 — 결제 성공·주문번호 표시 (SCR-008)
12. 추가 제안 — 장바구니 하단 추천 메뉴 (SCR-005)
13. 영수증 출력 — 영수증 출력 여부 선택·프린터 (SCR-020, SC-015, Phase 2)
14. 접근성 설정 — 글자 크기·고대비 등 (SCR-014, SC-013)
15. 언어 전환 — 한국어/영어 UI 전환 (FWD-UI-003)
16. 이전·뒤로가기 — 단계 되돌리기·취소 (공통 네비게이션)

금지 요소:
테두리, 아이콘 박스 테두리, 외곽 박스, 그림자, 드롭 섀도우, 내부 그림자, 뉴모피즘, 3D 렌더링, 사실적인 사진 스타일, 실제 음식 사진, 텍스트, 글자, 숫자, 워터마크, 로고, 브랜드명, 불균일한 여백, 불균일한 간격, 어긋난 그리드, 서로 다른 아이콘 크기, 복잡한 배경, 흐릿한 이미지, 낮은 해상도, 과도한 디테일, 그라데이션 남용, 유리·글래스모피즘 효과.

출력 형식:
고해상도 벡터 스타일 아이콘 시트.
4x4 그리드.
동일한 상하좌우 여백.
동일한 행/열 간격.
테두리와 그림자 없는 깔끔한 플랫 아이콘 팩.
프레시 그린(#16A34A)과 크림(#FFFDF3) 톤의 ASAK Salady 키오스크 무드.
```

## 사용 방법

### Figma AI (권장 워크플로)

1. `kiosk_design` 파일에서 **DS-01 Fresh Greens** 또는 **DS-02 Modern Minimal** 프레임 옆에 빈 프레임을 만듭니다.
2. Version A 또는 B 프롬프트 블록 전체를 복사해 Figma AI 이미지 생성에 붙여넣습니다.
3. 생성된 4×4 시트를 **Vectorize** 또는 수동 트레이싱 후, 16개 아이콘을 개별 컴포넌트로 분리합니다.
4. DS Variables(`ds-candidate-a-fresh-greens` / `ds-candidate-b-modern-minimal`)의 primary·accent 색으로 아이콘 fill을 맞춥니다.

### Midjourney / DALL·E

- 프롬프트 맨 앞에 `--style raw` 또는 `flat vector icon sheet, no text`를 추가하면 텍스트 유입을 줄일 수 있습니다.
- 해상도: **2048×2048** 이상 권장 (4×4 그리드 분할 후 512×512 셀당 아이콘 추출).
- 생성 후 Photoshop·Figma에서 4등분 그리드로 슬라이스해 개별 PNG/SVG로보냅니다.
- 텍스트·워터마크가 생기면 재생성하거나 수동 제거합니다.

### DS-02 컴포넌트 플러그인 연계

- 플러그인이 만드는 DS-01~07 프레임의 **Components** 섹션(834×2400)에 아이콘 시트를 배치해 DS 후보별 톤을 한눈에 비교할 수 있습니다.
- 플러그인 README: [figma-create-ds02-components-plugin/README.md](../figma-create-ds02-components-plugin/README.md)
- 아이콘은 플러그인 자동 생성 컴포넌트(카테고리 탭·결제 row·옵션 칩 등)와 **동일 팔레트**를 쓰면 SCR 와이어프레임과 시각적으로 일치합니다.

### 후처리 체크리스트

- [ ] 16개 아이콘 크기·선 굵기 균일
- [ ] 텍스트·숫자·로고 없음
- [ ] 44×44px 터치 타깃에서 실루엣 인식 가능
- [ ] DS 토큰 hex와 fill 색 일치 ([candidate-B](../kiosk-design-system-candidate-B.md))
- [ ] SCR-001~008 핵심 주문 흐름 아이콘(1~12번) 우선 Figma 컴포넌트화
