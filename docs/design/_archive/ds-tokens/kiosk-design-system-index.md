# ASAK 키오스크 디자인 시스템

> **정본**: Candidate B — **DS-02 Modern Minimal**
>
> 회의 결정(2026-07-06)에 따라 다른 디자인 후보와 비교용 하이브리드 문서는 정리했다.

## 선택 기준

- 대형 타이포와 높은 대비로 키오스크 가독성을 확보한다.
- Charcoal `#1A1C20`과 Electric Lime `#C8F135`을 핵심 색으로 사용한다.
- 카드·버튼은 흰색 표면과 단순한 사각 radius를 기본으로 한다.
