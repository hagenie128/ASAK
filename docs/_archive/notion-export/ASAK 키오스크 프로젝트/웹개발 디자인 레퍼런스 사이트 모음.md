| [Pinterest](https://www.pinterest.com/) | 무드보드, 스타일 수집 | 분위기별 이미지 저장, 컬러/브랜딩 방향 잡기 |

## 8. 작업 단계별 추천 루틴

### 1단계: 방향 잡기

프로젝트 초반에는 너무 세부 UI부터 보지 말고 전체 무드와 구조를 먼저 잡습니다.

- Pinterest
- Savee
- Awwwards
- Siteinspire
- [curated.design](http://curated.design)

### 2단계: 화면 구조 잡기

랜딩페이지나 웹서비스 화면을 만들 때는 섹션 흐름을 먼저 봅니다.

- [landing.love](http://landing.love)
- [saaspo.com](http://saaspo.com)
- [navbar.gallery](http://navbar.gallery)
- [cta.gallery](http://cta.gallery)
- [checklist.design](http://checklist.design)

### 3단계: UI 패턴 구체화

로그인, 장바구니, 결제, 온보딩, 마이페이지처럼 기능 단위 화면을 참고합니다.

- Mobbin
- Component Gallery
- Checklist Design
- Spotted in Prod

### 4단계: 인터랙션/디테일 추가

화면이 너무 밋밋할 때는 모션과 마이크로 인터랙션을 확인합니다.

- Spotted in Prod
- App Motion
- Awwwards
- CSS Design Awards

### 5단계: 포트폴리오로 정리

완성된 작업물을 보여줄 때는 결과물뿐 아니라 과정, 문제 해결, 역할을 같이 보여줍니다.

- Behance
- Cargo
- Framer
- Readymag
- e-jungle Portfolio
