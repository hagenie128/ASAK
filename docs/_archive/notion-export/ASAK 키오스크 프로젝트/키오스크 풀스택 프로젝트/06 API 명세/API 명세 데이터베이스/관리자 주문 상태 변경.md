## 처리 기준

- 주문 상태값은 `common_code`의 주문 상태 코드 기준을 따른다.
- Week 5~6 MVP 상태 흐름은 `RECEIVED → PREPARING → COMPLETED`로 제한한다.
- 상태 변경은 관리자 주문 상세 화면에서 수행한다.
- 결제 실패 또는 취소 흐름은 확장 범위로 분리한다.
