# 핵심 화면 전환 4단계 이내로 단순화

검수 기준: 주문 시작, 메뉴 선택, 옵션 선택, 장바구니/결제 흐름이 불필요한 중복 화면 없이 이어진다.
구분: 화면
기능계층: 기본기능
단계: FWD
비고: 키오스크 사용성 기준으로 화면 설계와 함께 검토한다. (2026-07-04 베미 확인) 여기서 '4단계'는 결제 직전(주문확인)까지만지정한 범위임: 메뉴선택→옵션선택→장바구니→주문확인(4개 화면). 결제완료까지 범위는 종합기획서/TC-002 기준(5개 화면, 결제 포함)을 따로 사용한다 — 둘은 서로 다른 구간을 재는 것이라 충돌이 아니다.
상세 설명: 사용자가 주문 시작부터 결제 전 확인까지 과도한 화면 이동 없이 진행할 수 있도록 핵심 흐름을 단순화한다.
상태: 예정
요구사항 ID: FWD-UI-002
우선순위: 중
