# 신규 고객의 기본 주문 흐름 성공

관련 API: GET /api/menus, POST /api/orders
관련 화면: 메뉴선택, 옵션선택, 결제
구분: 통합
기대 결과: 결제 완료 후 주문번호가 화면에 표시됨
단계: KSD
비고: SC-001 연결
상태: 예정
수행 절차: 1) 키오스크 초기화면 진입 2) ASAK 대표 메뉴 1개 선택 3) 추천조합 기본값 확인 후 토핑 1개 추가 4) 장바구니 담기 5) 결제수단 선택 후 결제
오류 발생: N
우선순위: 상
전제조건: 키오스크 정상 구동, 모든 메뉴 판매중
테스트 ID: TC-001
테스트 데이터: ASAK 대표 메뉴 1개 + 토핑 1개
