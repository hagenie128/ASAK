# 메뉴 선택

관련 API: API-001 GET /api/categories, API-002 GET /api/menus
관련 시나리오: 고객 주문 시나리오
관련 요구사항: FWD-UI-001, FWD-UI-002
관련 테스트: TC-001, TC-002
구분: 키오스크
단계: FWD
도메인: 식당
비고: 핵심 재료/베이스 재료 품절 시 메뉴 주문 불가. 일반 기본 재료 품절 시 메뉴 카드에 재료 품절 뱃지 표시
