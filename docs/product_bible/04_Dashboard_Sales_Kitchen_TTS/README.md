# ASAK Product Bible Pack 4

## Scope

이 Pack은 관리자 운영 경험의 중심이 되는 네 기능을 다룬다.

1. Dashboard
2. Sales
3. Kitchen / Live Order
4. TTS

## Operating Flow

```text
Dashboard
→ 진행 중 주문 확인
→ Kitchen / Live Order 처리
→ 주문 완료
→ TTS 호출
→ Sales 집계
```

- Dashboard는 운영 판단용이다.
- Kitchen은 실시간 처리용이다.
- Sales는 사후 분석용이다.
- TTS는 완료 상태의 부가 알림이다.
