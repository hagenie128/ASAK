# ASAK Product Bible Pack 7 — Screen Bible (화면 정본)

- Screen Registry (화면 레지스트리)

- 관리자 화면
- 확장 화면

각 문서는 다음을 포함한다.

- 목적

- 주요 데이터
- 필수 상태
- 제품 규칙
- React 컴포넌트 맵
- API 계약
- 예외 케이스
- 접근성

Screen ID가 충돌하면 `SCREEN_REGISTRY.md`를 먼저 수정한다.
