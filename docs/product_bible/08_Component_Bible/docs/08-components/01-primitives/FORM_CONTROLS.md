# Form Controls

## Includes

- Input
- SearchInput
- Checkbox
- Radio
- Switch
- Select

## Rules

- label 필수
- error message 연결
- placeholder는 label 대체 불가
- Switch는 즉시 반영 여부 명확히
- Checkbox는 다중 선택
- Radio는 단일 선택

## Admin Use

- IngredientSelectModal
- Login
- MenuForm
- FilterBar
- Payment settings
