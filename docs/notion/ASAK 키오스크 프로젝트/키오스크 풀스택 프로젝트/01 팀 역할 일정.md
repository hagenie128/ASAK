## 팀 정보

> ✅ 현재 팀은 이하진·김나연 2인입니다. 역할은 기능 경계가 아니라 실제 구현 흐름을 기준으로 나누고, 화면·API 연동은 함께 검토합니다.
>

| 이름 | 역할 | 담당 기능 | GitHub ID | 상태 |
| --- | --- | --- | --- | --- |
| 이하진 | 백엔드·DB | 메뉴/옵션·주문 API, DB·시드·통계 View 관리 | hagenie128 | 확정 |
| 김나연 | 프론트엔드·관리자 | 키오스크·관리자 화면, 결제/관리자 API 연동, 문서·발표 | nayeon0828 | 확정 |

## 역할 운영 원칙

- **이하진**: DB 구조와 메뉴·주문 도메인, 조회/생성 API를 담당한다.
- **김나연**: 고객·관리자 화면과 결제/관리자 흐름의 연동 및 문서화를 담당한다.
- API 응답·DB 변경·화면 변경은 두 사람이 함께 확인한다.

## 협업 시 유의할 점

- 막히는 작업은 1시간 안에 공유하고, DB/API 계약 변경은 작업 전에 합의한다.
- 데일리 워크로그에 진행·막힘·다음 작업을 남긴다.

## 체크인 · 협업 리듬

- **매주 금요일**: 스프린트 데모 + 회고 (전원)
- **스프린트 시작일**: 프론트 1 + 백엔드 1 API 싱크 (명세 변경 확인)
- 일일 진행은 Git [`worklog/daily/`](https://github.com/hagenie128/ASAK/tree/main/worklog/daily) + Notion [📅 일일 워크로그](../%F0%9F%93%85%20%EC%9D%BC%EC%9D%BC%20%EC%9B%8C%ED%81%AC%EB%A1%9C%EA%B7%B8%20eeae4beb07ad4051928a87de0ea4c8f9.md) 캘린더로 공유
