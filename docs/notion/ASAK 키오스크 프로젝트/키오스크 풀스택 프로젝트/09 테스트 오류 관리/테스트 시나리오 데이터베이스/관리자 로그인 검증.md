# 관리자 로그인 검증

관련 API: 관리자 로그인 API(후반 확장)
관련 화면: SCR-015
구분: 관리자
기대 결과: 정상 계정은 관리자 화면으로 이동하고, 잘못된 계정은 오류 메시지를 표시한다.
단계: LMIS
비고: role_id 기반 권한 확인 포함
상태: 예정
수행 절차: 1. 관리자 로그인 화면에 진입한다. 2. 정상 계정으로 로그인한다. 3. 잘못된 계정으로도 시도한다.
오류 발생: N
우선순위: 중
전제조건: 관리자 계정과 role_id가 준비되어 있음
테스트 ID: TC-009
테스트 데이터: admin 계정, invalid 계정
