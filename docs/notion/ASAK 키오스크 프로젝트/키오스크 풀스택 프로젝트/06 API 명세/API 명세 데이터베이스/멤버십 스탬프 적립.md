## 요청 예시

```json
POST /api/membership/stamps
{
  "orderId": 1,
  "memberId": 10,
  "confirmStamp": true
}
```

> 결제 완료(orderId=1, ASAK-20260703-001) 후 1회 확인. 미가입 고객은 confirmStamp=false로 스킵.
