# 관리자 메뉴 등록/수정

API ID: API-012
Error: ADMIN_MENU_SAVE_FAILED, MENU_REQUEST_INVALID
Method: POST
Request: Body: name, categoryId, price, imageUrl, description, ingredients, optionGroups, recommendedOptions, tags
Response: { success, status, code, message, data: { menuId } }
URL: /api/admin/menus
관련 테이블: menu, menu_ingredient, menu_option_group, menu_option, menu_tag
구분: 관리자
사용 화면: 관리자 메뉴 등록/수정
상태: 예정
설명: 관리자가 신규 메뉴를 등록하거나 메뉴 기본정보를 수정한다.
인증: N
처리 내용: 메뉴 기본정보와 기본 재료, 추천 드레싱(menu_option), 태그 연결을 함께 저장한다. 실제 구현은 Week 6(SCR-009~011) 이후로 둔다.
Related to 테스트 시나리오 데이터베이스 (↔ API): 관리자 메뉴 등록/수정 검증 (../../09%20%ED%85%8C%EC%8A%A4%ED%8A%B8%20%EC%98%A4%EB%A5%98%20%EA%B4%80%EB%A6%AC/%ED%85%8C%EC%8A%A4%ED%8A%B8%20%EC%8B%9C%EB%82%98%EB%A6%AC%EC%98%A4%20%EB%8D%B0%EC%9D%B4%ED%84%B0%EB%B2%A0%EC%9D%B4%EC%8A%A4/%EA%B4%80%EB%A6%AC%EC%9E%90%20%EB%A9%94%EB%89%B4%20%EB%93%B1%EB%A1%9D%20%EC%88%98%EC%A0%95%20%EA%B2%80%EC%A6%9D.md)

## 요청 예시

```json
POST /api/admin/menus
{
  "categoryId": 231,
  "name": "스파이시 쉬림프 샌드위치",
  "price": 8900,
  "imageUrl": "/assets/menu/364.png",
  "description": "케이준 쉬림프, 할라피뇨, 토마토...",
  "optionGroupIds": [240, 244, 245]
}
```
