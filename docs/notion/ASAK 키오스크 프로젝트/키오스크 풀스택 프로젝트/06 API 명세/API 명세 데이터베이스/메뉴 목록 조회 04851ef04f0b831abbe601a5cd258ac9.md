
## 처리 기준

- 카테고리별 메뉴 목록을 조회한다.
- 핵심 재료 또는 베이스 재료가 품절이면 메뉴를 품절로 표시한다.
- 일반 기본 재료가 품절이면 메뉴 자체 품절이 아니라 `soldOutBadges`로 별도 표시한다.
- 목록 화면에서는 상세 옵션 전체를 내려주지 않고 메뉴 카드에 필요한 정보만 내려준다.
