# 관리자 결제수단 설정 변경

API ID: API-014
Error: PAYMENT_METHOD_UPDATE_FAILED
Method: PATCH
Request: Body: isActive, sortOrder
Response: { success, status, code, message, data: { methodId, isActive, sortOrder } }
URL: /api/admin/payment-methods/{methodId}
관련 테이블: payment_method_config, common_code
구분: 관리자
사용 화면: 관리자 결제수단 설정
상태: 예정
설명: 관리자가 결제수단 노출 여부와 정렬 순서를 변경한다.
인증: N
처리 내용: 결제수단은 Boolean으로 단순화하지 않고 향후 카드/간편결제/현금 등 확장 가능하도록 설정 테이블로 관리한다.
