# 관리자 판매 항목 목록 조회

API ID: API-010
Error: SOLD_OUT_ITEM_LIST_FAILED
Method: GET
Request: Query: targetType(MENU|INGREDIENT|OPTION_ITEM), keyword(optional)
Response: { success, status, code, message, data: [{ targetType, targetId, name, isSoldOut, reasonType }] }
URL: /api/admin/sold-out-items
관련 테이블: menu, ingredient, option_item, menu_ingredient, common_code
구분: 관리자
사용 화면: 관리자 판매 항목 품절 관리
상태: 예정
설명: 관리자가 품절 처리할 수 있는 메뉴/재료/옵션항목 목록을 조회한다. GET /api/admin/sold-out-items(조회, 본 API)와 PATCH /api/admin/sold-out-items(변경, API-009)는 같은 경로를 Method로 구분한다.
인증: N
처리 내용: 품절 변경 API(API-009) 전 단계로, 관리자가 변경 가능한 대상을 목록으로 확인한다. CORE/BASE/DEFAULT/OPTION_ITEM 기준을 함께 내려준다.
