# 장바구니 검증

API ID: API-016
Error: CART_INVALID, CART_VALIDATE_FAILED
Method: POST
Request: Body: items[{ menuId, quantity, optionItems, excludedIngredientIds }]
Response: { success, status, code, message, data: { valid, invalidItems, totalPrice } }
URL: /api/cart/validate
관련 테이블: menu, ingredient, option_item, menu_ingredient, common_code
구분: 키오스크
사용 화면: 장바구니, 주문 확인
상태: 예정
설명: 프론트 장바구니에 담긴 메뉴/옵션이 현재도 주문 가능한지 검증한다.
인증: N
처리 내용: 장바구니 자체는 프론트 상태로 관리하되, 주문 생성 전 품절/가격 변경 여부를 확인한다.

## 요청 예시

```json
POST /api/cart/validate
{
  "items": [
    {
      "menuId": 364,
      "quantity": 1,
      "optionItems": [{ "optionItemId": 269, "quantity": 1 }],
      "excludedIngredientIds": [169]
    }
  ]
}
```
