## React 18 다운그레이드 (기존 프로젝트)

```jsx
npm install react@18 react-dom@18
```

💡 **혹시 빨간색 에러(Peer Dependency Conflict)가 나나요?**

```jsx
npm install react@18 react-dom@18 --force
```

## React 18 새 프로젝트 생성

```jsx
npx create-react-app 프로젝트이름 --template cra-template
cd 프로젝트이름
npm install react@18 react-dom@18 --force
```

설치 후 `package.json`의 `dependencies` 예시:

```
"dependencies": {
  "react": "^18.3.1",
  "react-dom": "^18.3.1"
}
```
