# 강민준

AI활용정도: 전체 로직까지 생성
가장편했던툴: VS Code, IntelliJ
강점: 현시점 판단 어려움, 열심히 따라가겠음
기여가능한경험: 팀플 경험 많아 방해되지 않을 자신, 다만 웹 부분은 처음
기존키오스크개선점: 포인트적립/혜택 여부를 결제 전에 미리 결정하게 해서 결제 도중 끊기지 않게 (REQ-009와 일치)
별로였던키오스크: UI 인식이 별로였던 키오스크(결제버튼 위치 이상하거나 복잡한 경우)
사용AICLI코딩툴: Cursor, Copilot
사용AI웹서비스: ChatGPT
새로배우고싶은기술: 어느 한쪽이라도 중급 정도 숙지
선호IDE: VS Code, IntelliJ
선호역할: 상관없음
일정충돌예정: 남은 일정간 무리없음
자신없는부분: 0에서 1을 처음 만들 때
좋았던키오스크: 이용자 편의를 먼저 고려한 단순한 UI, 결제 안될 시 데스크에서 결제 가능한 곳
주당투입가능시간: 5-10시간
프로젝트기대하는것: 맡은 파트 및 작업 부분을 제대로 숙지하는 것
피드백선호: 직설적 피드백 OK
학습_보완하고싶은영역: 전체적으로 전 파트
학습_어려웠던부분: 새로운 메소드, 구문 암기 등 새 파트 나갈 때마다
학습_잘맞았던부분: 배운 후 단기간 내 복습/실습하면 비교적 수월했음
협업스타일: 혼자하다가 정 안되면 질문
흥미느낀쪽: 둘 다 비슷
