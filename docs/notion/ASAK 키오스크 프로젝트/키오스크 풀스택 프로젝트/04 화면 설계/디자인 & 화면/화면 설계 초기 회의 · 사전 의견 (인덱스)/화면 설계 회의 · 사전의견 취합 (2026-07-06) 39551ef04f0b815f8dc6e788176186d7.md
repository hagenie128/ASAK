## 합의 가능 / 회의에서 결정할 것

### 합의 가능 (의견 겹침·충돌 적음)

- 결제 **대기 시 로딩** 화면 표시 (박유진) 
- 통과
- 결제 **실패·재시도**는 별도 화면보다 팝업·토스트 (박유진) 
- 통과
- DS 후보 **DS-02·04** — 2인 이상 교차 선택
- 관리자 화면은 **가독성·정보 밀도** 우선 (강민준, 김나연 SCR-019)

### 회의에서 결정할 것

- SCR **개수·목록** 확정 (18 vs 21, 홈+포장/매장 추가 여부) 
- 더 많아짐
- **장바구니 ↔ 주문확인** — 통합·Depth 축소 vs 분리·정보 확장 
- 축소(추천 제안 상품 하단 추가)
- **DS 최종** — **DS-02 Modern Minimal** 채택 (DS-08 02+04 하이브리드 논의·미채택 참고안)
- **고객/관리자 DS 분리** — 별도 톤 가이드 여부
- **추천 메뉴** — MVP 포함 vs Phase 2
- 후순위
- **박유진 DS 1순위** — DS-01 vs DS-04 재투표
- 4
