# 관리자 주문 관리

Figma 링크: https://www.figma.com/design/iqaoVwFjFE6Zq1WpOVgjeG/kiosk_design?node-id=75-17
개발 완료: No
관련 API: API-007 GET /api/admin/orders
관련 시나리오: 관리자 주문 목록 확인 / 주문 상세 확인 시나리오
관련 요구사항: LMIS-ORDER-001, LMIS-ORDER-002, LMIS-ORDER-004
관련 테스트: TC-014
구분: 관리자
단계: KSD
도메인: 식당
비고: 주문 상태 변경은 관리자 주문 상세 화면에서 수행한다.
상태: 와이어프레임
우선순위: 상
입력 데이터: status
화면 ID: SCR-009
화면 설명: 관리자가 주문 목록을 확인하고 특정 주문의 상세 화면으로 이동하는 화면
출력 데이터: 주문 목록, 주문번호, 주문유형, 결제상태, 주문상태, 주문금액, 주문일시
