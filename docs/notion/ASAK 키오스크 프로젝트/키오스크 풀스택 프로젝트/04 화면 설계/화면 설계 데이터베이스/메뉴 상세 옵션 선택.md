# 메뉴 상세 / 옵션 선택

Figma 링크: https://www.figma.com/design/iqaoVwFjFE6Zq1WpOVgjeG/kiosk_design?node-id=75-4
개발 완료: No
관련 API: API-003 GET /api/menus/{menuId}, API-004 GET /api/menus/{menuId}/options
관련 시나리오: 옵션 선택 시나리오
관련 요구사항: FWD-MENU-001, FWD-MENU-002, FWD-MENU-003, FWD-MENU-004, FWD-MENU-005, FWD-MENU-015, LMIS-MENU-002
관련 테스트: TC-003
구분: 키오스크
단계: FWD
도메인: 식당
비고: 기본 포함 재료도 추가 토핑으로 선택 가능. 옵션 재료 품절은 해당 옵션칸만 품절. 알레르기·칼로리는 장바구니가 아닌 본 화면(옵션 선택)에 표시 (2026-07-06).
상태: 와이어프레임
우선순위: 상
입력 데이터: menuId, 기본 재료 목록, excludedIngredientIds, optionItems(optionItemId, ingredientId, quantity)
화면 ID: SCR-004
화면 설명: 메뉴 상세 정보를 확인하고 기본 재료 제외, 드레싱/토핑 등 옵션을 선택하는 화면입니다. 알레르기·칼로리 정보는 이 화면에서 표시합니다.
출력 데이터: 선택 메뉴, 기본 재료, 제외 재료, 품절 재료 뱃지, 추가 옵션, 옵션 수량, 옵션그룹 유형, 세트 옵션, 옵션 제공량/단백질, 옵션 아이콘/색상, 품절 옵션칸, 베이스 변경 가능 여부, 예상 금액, 예상 알레르기 표시
