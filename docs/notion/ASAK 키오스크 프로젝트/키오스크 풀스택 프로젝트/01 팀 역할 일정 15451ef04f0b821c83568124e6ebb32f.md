## 역할 배정 근거 (사전조사 기반)

- **박유진**: 리액트 학습에 흥미, Figma 경험 있어 UI 쪽 강점. 화면 구현에 집중.
- **강민준**: 웹 개발은 처음이지만 실제 사용자 흐름을 보며 아이디어를 내는 데 강점이 있음 → 장바구니/결제 화면 구현과 주문 흐름 확인 보조를 함께 담당.
- **이하진**: 백엔드 흥미, 데이터 관리 학습이 잘 맞음, 요식업 경험으로 도메인 이해도 높음.
- **김나연**: Spring Boot·자바 자신감, 팀 조율 경험 풍부(전 직장에서 팀원간 트러블 해결 경험) → 문서화·발표자료까지 총괄.

## 페어링

- **프론트 페어**: 박유진 + 강민준
- **백엔드 페어**: 이하진 + 김나연

## 협업 시 유의할 점

- 이하진: 막히면 오래 혼자 붙잡는 경향 → "1시간 넘으면 바로 질문" 규칙 권장. 최근 출결 관리에 스스로 신경쓰겠다고 했으니 데일리 체크인에서 가볍게 서로 챙겨주면 좋음.
- 박유진만 부드러운 피드백 선호, 나머지는 직설적 OK.
- 김나연·강민준이 팀플 경험 풍부 → 초반 팀워크는 이 둘이 자연스럽게 리드.

## 체크인 · 협업 리듬

- **매주 금요일**: 스프린트 데모 + 회고 (전원)
- **스프린트 시작일**: 프론트 1 + 백엔드 1 API 싱크 (명세 변경 확인)
- 일일 진행은 Git [`worklog/daily/`](https://github.com/hagenie128/ASAK/tree/main/worklog/daily) + Notion [📅 일일 워크로그](../%F0%9F%93%85%20%EC%9D%BC%EC%9D%BC%20%EC%9B%8C%ED%81%AC%EB%A1%9C%EA%B7%B8%20eeae4beb07ad4051928a87de0ea4c8f9.md) 캘린더로 공유
