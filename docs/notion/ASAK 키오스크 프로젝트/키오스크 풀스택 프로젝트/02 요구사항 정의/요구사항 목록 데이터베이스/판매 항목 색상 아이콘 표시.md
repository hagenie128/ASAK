# 판매 항목 색상/아이콘 표시

검수 기준: 등록된 색상 또는 아이콘 정보가 메뉴 상세와 옵션 선택 화면에 표시된다.
구분: 화면
기능계층: 추가기능
단계: FWD
비고: option_item의 icon_url, color_hex 확장 컴럼과 연결 가능하다. 단, 재료(ingredient) 자체에는 색상/아이콘 컴럼이 없으므로, 재료 기반 표시가 필요하면 option_item 칸럼을 경유해야 한다.
상세 설명: 재료, 옵션, 카테고리 등 판매 항목에 색상 또는 아이콘 이미지를 연결해 화면에서 식별성을 높인다.
상태: 예정
요구사항 ID: FWD-UI-004
우선순위: 하
