# 세트 구성 관리

검수 기준: 세트는 사이드와 음료를 각각 선택할 수 있고, 각 선택 항목의 추가 금액 또는 할인 전 금액이 주문 화면과 API에 표시될 수 있다.
구분: 관리자
기능계층: 추가기능
단계: LMIS
비고: Week 5 MVP 범위에서는 구현하지 않고, 9주 후반(Week 7~8) 옵션기능 확장으로 관리한다. DB는 SET_SIDE/SET_DRINK 옵션그룹과 option_item.original_price/extra_price로 확장 가능하게 둔다.
상세 설명: 관리자는 메인 메뉴, 사이드, 음료를 묶은 세트 구성을 관리할 수 있어야 한다.
