# 브랜드 방향

## 확정 기준

- 서비스명은 **ASAK / 아삭**으로 사용한다.
- ASAK은 **A SALAD A KIOSK**의 약자이면서, 신선한 채소의 아삭한 식감을 함께 떠올리게 하는 이름으로 정리한다.
- 실제 브랜드를 그대로 재현하지 않고, 건강식 키오스크를 위한 가상 브랜드로 구성한다.

## 핵심 이미지

- 신선함
- 아삭함
- 건강함
- 친근함
- 건강메이트
- 편의성

## 피해야 할 방향

- 과한 자연풍
- 고급스럽고 무거운 분위기
- 웅장한 브랜드 느낌
- 동남아 휴양지 느낌
- 촌스러운 초록색
- 복잡한 화면
- 메뉴 선택이 어려운 구조

# 디자인 방향

## 컬러 기준

| 역할 | 색상 | 용도 |
| --- | --- | --- |
| Primary Green | #16A34A | 메인 버튼, 선택 상태, 주요 액션 |
| Primary Dark | #15803D | hover/pressed, 강조 텍스트 |
| Fresh Lime | #A3E635 | 신선함 포인트, 가벼운 배지 |
| Crunch Yellow | #FACC15 | 포인트 배지, 추천 표시, 선택 보조 |
| Cream Background | #FFFDF3 | 전체 배경, 부드러운 화면 톤 |
| Text Main | #1F2937 | 본문/제목 텍스트 |

## UI 분위기

- 메뉴 이미지는 적절히 사용하되 화면을 복잡하게 만들지 않는다.
- 메뉴는 직관적인 카드형 UI로 구성한다.
- 옵션 선택 화면은 단계와 선택 상태가 바로 보이게 만든다.
- 필수 버튼은 크고 명확하게 배치한다.
- 고객용 화면은 단순하게 유지한다.
- 관리자 화면은 표, 상태, 필터 중심으로 구성한다.
