키오스크 ui 디자인 레퍼런스 기반 → 화면별 프로토타입 디자인 화면 결정 및 수정

## 회의 결과

### 고객 키오스크 화면 설계

- 고객 키오스크 화면 → 세로 디자인

| No | 화면 | 결정 사항 |
| --- | --- | --- |
| 1 | 메뉴 선택 화면 | 메뉴 선택 시 하단에 선택 메뉴 영역 추가 |
| 2 | 메뉴 상세 / 옵션 선택 | 옵션 커스텀 상품은 옵션 선택 페이지로 이동 |
| 3 | 메뉴 상세 / 옵션 선택 | 단일 상품은 장바구니 페이지 생략 후 바로 담기 |
| 4 | 장바구니 & 주문 확인 | 레퍼런스 디자인 2개 요소 병합 |
| 5 | 결제창 | 가격 리스트는 펼쳐보기 형태로 제공 |
| 6 | 결제창 | 결제 수단 선택 중심의 UI 구성 |
| 7 | 주문 완료 | 영수증 + 주문번호 발급 버튼 추가 |
| 8 | 주문 완료 | 주문번호만 출력 버튼 추가 |

### 관리자 키오스크 화면 설계

- 관리자 키오스크 화면 → 가로 디자인

| 화면명 | 디자인 형식 | 주요 기능 / 특이사항 |
| --- | --- | --- |
| 관리자 주문 관리 화면 | 차트형 리스트 형식 | 완료/삭제 버튼 형식으로만 구성. 완료 처리 시 TTS 음성 알림 처리 |

| 관리자 결제 수단 설정 화면 | 카드형 디자인 | 토글 박스 or 체크박스 형식으로 설정 |
| 관리자 매출 요약 화면 | - | 매출 주별 요약 + 시간대별 매출 현황으로 정리 |
