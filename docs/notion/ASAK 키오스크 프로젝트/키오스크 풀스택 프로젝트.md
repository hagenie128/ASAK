## 산출물 DB 연동 규칙

요구사항 → 시나리오 → 화면 → API → WBS → 테스트는 **텍스트 ID**로 상호 참조합니다.

| DB | 키 필드 | 연결 대상 예시 |
| --- | --- | --- |
| 요구사항 목록 | `요구사항 ID` (예: FWD-UI-001) | 시나리오 `관련 요구사항` |
| 사용자 시나리오 | `시나리오 ID` (예: SC-001) | 테스트 TC-001~013 `관련 시나리오` |
| 화면 설계 | 화면명 / 화면 ID | API `사용 화면` · 시나리오 `관련 화면` |
| API 명세 | `API ID` (예: API-001) | 시나리오 `관련 API` · WBS `관련 산출물` |
| WBS | `작업 ID` (예: WBS-010) | 요구사항 상태 · 체크리스트 |

> **상태 불일치 주의**: 요구사항 DB(`예정→테스트완료`)와 WBS DB(`예정→완료`)는 상태 옵션이 다릅니다. 마일스톤 완료 시 **양쪽을 함께** 갱신하세요.
>
