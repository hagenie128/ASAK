### 팀용 ASAK AI 스킬 설치 패키지 개선·배포 — `asak-agent-kit` · `tooling` / `docs`

- **작업 목적:** 팀원이 Codex 등 여러 AI 도구에서 같은 ASAK 안전 규칙과 단축 명령을 쉽게 설치·사용하도록 패키지화한다.
- **직접 구현:** 공통 스킬 원본과 도구별 패키지를 구성하고, Windows 설치/검증 도구와 쉬운 설치 안내를 개선했다. 복잡한 번호 입력·경로 붙여넣기 흐름을 AI 선택·폴더 선택 방식으로 바꾸고, `.bat`의 한글 깨짐 문제를 수정했다.
- **AI 도움:** Codex/ChatGPT에게 스킬 구조, 설치 흐름, 도구별 지침 변환, 사용 설명 초안을 요청했다. 설치 범위·안전 규칙·팀 배포 판단과 사용성 문제 발견·수정은 직접 진행했다.
- **이슈:** 기존 설치 안내가 복잡하고 `.bat`에서 한글 파일명을 직접 다뤄 글자가 깨짐 → 배치 파일 내부 영어 사용, PowerShell 안내, 단일 안내 파일 중심으로 개선했다.
- **검증:** `asak-agent-kit`의 `packages/`, `sources/skills/`, `installers/`, `docs/` 구조와 초기화 커밋을 확인했다. Git 배포는 사용자 확인 및 공유 대화의 v2.1 배포 기록을 근거로 남긴다.
- **포트폴리오 요약:** 반복적인 프로젝트 규칙을 AI 스킬과 설치 도구로 제품화해, 팀원이 사용하는 AI가 달라도 동일한 작업 원칙을 적용할 수 있게 했다.

## 블로커 / 공유 사항

- `FEATURE_LOOKUP`과 기능 매트릭스는 구현 안내 문서이며, 실제 API 또는 화면이 구현 완료되었다는 뜻은 아니다.
- 영수증·멤버십·쿠폰 등 확장 화면은 기준만 기록했으며 MVP 범위 포함 여부는 팀 결정이 필요하다.

## 내일 계획

- 구현 담당자가 선택한 Vertical Slice부터 가이드의 Screen ID·API·상태·QA 항목을 실제 코드와 대조한다.
- 이미지 자산은 메뉴/재료 ID 매핑과 라이선스·출처를 다시 확인한 뒤 화면에 연결한다.
- Agent Kit을 실제 팀원 환경에서 설치·검증해 도구별 안내와 업데이트 경로를 보완한다.
